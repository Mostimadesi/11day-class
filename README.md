# 11day-class

这是一个用于记录我 11 天课程学习过程的仓库。

仓库内容将以 `.py` 文件和 `.md` 文件为主，主要用于：

- 记录每天的代码练习与小项目成果
- 整理学习笔记、知识点总结与问题复盘
- 沉淀每个阶段的心得体会，方便后续回顾与完善

目前仓库仍在持续更新中，部分内容先搭建了结构框架，后续会根据实际学习进度逐步补充。

## 项目目标

通过 11 天的连续学习，完成从基础知识到实践输出的积累，尽量做到：

- 每天都有可见的学习产出
- 每天都保留简单的过程记录
- 用代码和文字同时沉淀学习成果

## 仓库内容说明

当前仓库会重点包含以下两类内容：

### 1. Python 代码文件

用于保存课程中的练习代码、案例代码和阶段性成果。

示例：

- `hello_world.py`：day1中的“编写第一行代码”，用于输出 `Hello, World!`

### 2. Markdown 学习文档

用于整理：

- 每日学习笔记
- 重点知识总结
- 错题、报错和问题排查记录
- 学习心得与阶段复盘

示例：

- `day01.md`：day1的学习笔记，记录当天学习内容、心得体会和问题排查
